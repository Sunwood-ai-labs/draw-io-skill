# v0.2.0 Release Notes

Published: March 16, 2026

GitHub Release: [v0.2.0](https://github.com/Sunwood-ai-labs/draw-io-skill/releases/tag/v0.2.0)

Compare: [v0.1.0...v0.2.0](https://github.com/Sunwood-ai-labs/draw-io-skill/compare/v0.1.0...v0.2.0)

`v0.2.0` builds on the initial `v0.1.0` release with shape-aware SVG linting, a published VitePress documentation site, bilingual release-note mirrors, and richer showcase assets for repository onboarding.

## Highlights

- Expanded `scripts/check-drawio-svg-overlaps.mjs` to detect `edge-rect-border` on large frames, `edge-shape-border` on supported non-rect shapes, `rect-shape-border` contacts, and shape-aware text overflow from companion `.drawio` metadata.
- Added dedicated regression fixtures and verifier scripts for large frame border overlap, non-rect shape border overlap, and shape-aware text overflow under [`fixtures/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/fixtures) and [`scripts/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/scripts).
- Added a public VitePress docs surface with English and Japanese guides, showcase pages, release-note mirrors, and GitHub Pages deployment.

## Tooling And Automation

- Expanded [`package.json`](https://github.com/Sunwood-ai-labs/draw-io-skill/blob/v0.2.0/package.json) with `docs:build`, `docs:preview`, and a repository-level `verify` script that combines Node checks, fixture assertions, Python syntax validation via `uv`, and the VitePress build.
- Updated [`.github/workflows/ci.yml`](https://github.com/Sunwood-ai-labs/draw-io-skill/blob/v0.2.0/.github/workflows/ci.yml) to run `npm run verify`, support manual dispatch, cache npm installs, and refresh the checkout/setup actions.
- Added [`.github/workflows/pages.yml`](https://github.com/Sunwood-ai-labs/draw-io-skill/blob/v0.2.0/.github/workflows/pages.yml) to build `docs/.vitepress/dist` and deploy the published docs to GitHub Pages.

## Docs And Assets

- Added the public docs structure under [`docs/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/docs), including getting started, workflow, architecture, export-and-lint, troubleshooting, showcase, and release-note pages in both locales.
- Added icon-rich showcase diagrams, a shape-focused review sample, a Japanese-localized structure diagram, and a site icon under [`assets/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/assets).
- Updated `README.md`, `README.ja.md`, and `SKILL.md` so the repository, docs site, showcase samples, and release-note mirrors point to the same workflow.

## Validation

- Verified this release locally with `npm run verify` on March 16, 2026.
- The validation scope includes the expanded fixture suite, Python syntax validation for `scripts/find_aws_icon.py`, and the VitePress production build.

## Version Note

- The Git tag `v0.2.0` is the canonical identifier for this repository release.
- `package.json` still reports `1.0.0` and the package remains `private`, so repository package metadata should not be treated as the published release version.
