# v0.2.0 リリースノート

公開日: 2026-03-16

GitHub Release: [v0.2.0](https://github.com/Sunwood-ai-labs/draw-io-skill/releases/tag/v0.2.0)

比較: [v0.1.0...v0.2.0](https://github.com/Sunwood-ai-labs/draw-io-skill/compare/v0.1.0...v0.2.0)

`v0.2.0` では、初回公開版 `v0.1.0` を土台に、shape-aware な SVG lint、公開 VitePress docs、英日リリースノートのミラー、そしてショーケース用アセットを強化しました。

## ハイライト

- `scripts/check-drawio-svg-overlaps.mjs` を拡張し、大きなフレームへの `edge-rect-border`、対応する非矩形 shape への `edge-shape-border`、`rect-shape-border`、さらに companion `.drawio` を使う shape-aware な text overflow 検出に対応しました。
- [`fixtures/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/fixtures) と [`scripts/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/scripts) に、大型フレーム境界、非矩形 shape 境界、shape-aware text overflow 向けの回帰 fixture と verifier script を追加しました。
- 英語・日本語のガイド、ショーケース、release mirror を持つ公開 VitePress docs と、GitHub Pages 配備フローを追加しました。

## ツールと自動化

- [`package.json`](https://github.com/Sunwood-ai-labs/draw-io-skill/blob/v0.2.0/package.json) に `docs:build`、`docs:preview`、`verify` を追加し、Node チェック、fixture 検証、`uv` 経由の Python 構文確認、VitePress build をまとめて実行できるようにしました。
- [`.github/workflows/ci.yml`](https://github.com/Sunwood-ai-labs/draw-io-skill/blob/v0.2.0/.github/workflows/ci.yml) を更新し、`npm run verify`、手動実行、npm キャッシュ、各種 setup action の更新を反映しました。
- [`.github/workflows/pages.yml`](https://github.com/Sunwood-ai-labs/draw-io-skill/blob/v0.2.0/.github/workflows/pages.yml) を追加し、`docs/.vitepress/dist` を GitHub Pages に配備できるようにしました。

## Docs とアセット

- [`docs/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/docs) 配下に、はじめに、workflow、architecture、export-and-lint、troubleshooting、showcase、release note を英日両対応で公開しました。
- [`assets/`](https://github.com/Sunwood-ai-labs/draw-io-skill/tree/v0.2.0/assets) に、アイコン中心のショーケース図、shape review 用サンプル、日本語版構成図、サイトアイコンを追加しました。
- `README.md`、`README.ja.md`、`SKILL.md` を更新し、リポジトリ説明、docs、ショーケース、release note ミラーの導線を揃えました。

## 検証

- 2026-03-16 時点で `npm run verify` をローカル実行して確認しました。
- この検証には、追加した fixture 群、`scripts/find_aws_icon.py` の Python 構文確認、VitePress 本番 build が含まれます。

## バージョン注記

- このリポジトリの公開バージョン識別子は Git tag `v0.2.0` です。
- `package.json` は引き続き `1.0.0` かつ `private: true` のため、npm package の公開版バージョンとしては扱わないでください。
